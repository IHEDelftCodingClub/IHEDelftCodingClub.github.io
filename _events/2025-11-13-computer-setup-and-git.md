## Overview

Join us for a hands-on workshop where we set up our computers together, and start interacting with the command line and version control. Also, set up your membership profile for the Coding Club.

## What You'll Learn

- Setup your Development Environment
- Learn how to access the command line
- Learn about the power of version control
- Start interacting with Git and Github
- Set up your membership profile

## Prerequisites

- No prior experience required
- Bring a laptop
